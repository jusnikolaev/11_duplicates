# Anti-Duplicator

Find and remove duplicates in directory

# How use

<pre>
    <code>
    python3 duplicates.py --path 'path'
    </code>
</pre>


# Example

<pre>
    <code>
    python3 duplicates.py --path 'Users/user/Desktop'
    </code>
</pre>
